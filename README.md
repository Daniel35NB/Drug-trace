# Drug-trace

药物追踪系统 - 用于追踪和管理药物供应链的项目。

## 项目简介

本项目旨在提供一个可靠的药物追踪解决方案，确保药品从生产到消费的全程可追溯性。

## 功能特性

- 📦 药物批次管理
- 🔍 供应链追踪
- 📊 数据可视化
- 🔐 安全认证
